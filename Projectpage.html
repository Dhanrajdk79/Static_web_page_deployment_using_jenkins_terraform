Project: Static Website CICD

This repository demonstrates: Terraform provisioning of an EC2 instance + Nginx, and Jenkins pipeline to deploy updates via SSH.

How it works

  1. Terraform provisions server & security group
  2. UserData installs nginx, git and clones this repo
  3. Jenkins pipeline runs on push: SSH -> git pull -> restart nginx
